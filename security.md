## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in this project, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to [gibmepreo@gmail.com](mailto:gibmepreo@gmail.com).

You should receive a response within 48 hours. If for some reason you do not, please follow up via email to ensure we received your original message.

## Policy

We will investigate all legitimate reports and do our best to quickly fix the problem. We ask that you follow the [Responsible Disclosure](https://en.wikipedia.org/wiki/Responsible_disclosure) guidelines.
